About This Application

Built with Python

This web application is built using Python and Flask framework, providing a robust backend for task management.

Modern UI

Features a clean, responsive design with modern CSS styling and Font Awesome icons for an intuitive user experience.

RESTful API

Includes a complete REST API for task operations (CRUD) that can be easily extended or integrated with other applications.

Responsive Design

Fully responsive design that works seamlessly across desktop, tablet, and mobile devices.

Technology Stack

  • Backend: Python, Flask
  • Frontend: HTML5, CSS3, JavaScript
  • Icons: Font Awesome
  • Architecture: MVC Pattern

API Endpoints

GET /api/tasks Get all tasks
POST /api/tasks Create new task
PUT /api/tasks/:id Update task
DELETE /api/tasks/:id Delete task
